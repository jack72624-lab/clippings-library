Build $10,000 Websites using Claude Code

主題：用「設計干預」把 AI 從模板生成器變成高價量身網站
能力 · Pattern五點拼法
  1. 01

    八大支柱：觀點/排版/色彩、層次/圖像/文案、動效/行動端/隱形細節，逐項對照才做得出「昂貴感」。

  2. 02

    裝設計護欄 skill：Frontend Design 過濾掉 Inter 這類爛俗選擇；UI/UX Pro Max 給 57 風格、95 配色可挑。

  3. 03

    字體與文案節制：標題換 Fraunces、內文換 Geist；文案走節制路線（「Six dishes, one fire」）。

  4. 04

    打磨指令：「讓這區塊更昂貴精緻」會自動加膠卷顆粒、延遲隨動光暈（lagging motion）。

  5. 05

    部署陷阱：壓縮要選「資料夾內所有檔案」而非整個資料夾，選錯路徑多一層、載入空白。

一句話

$10K 網站的差別不在程式碼量，在品味與質感，靠設計護欄 + 高品質 brief + 打磨 pass 做出來。

能力 · Pattern四點拼法

  3. 03
